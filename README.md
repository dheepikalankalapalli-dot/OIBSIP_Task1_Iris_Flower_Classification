# Iris Flower Classification 🌸

## Project Overview
This project focuses on building a Machine Learning model to classify Iris flowers into different species based on their measurements.

The Iris dataset contains three flower species:
- Setosa
- Versicolor
- Virginica

The model learns from flower measurements and predicts the correct species.

## Objective
To train a Machine Learning classification model that can accurately identify Iris flower species using input features.

## Dataset
The dataset contains the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Target)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Workflow
1. Import required libraries
2. Load and explore dataset
3. Perform data preprocessing
4. Split data into training and testing sets
5. Train classification model
6. Evaluate model performance
7. Predict flower species

## Algorithms Used
- Logistic Regression
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)

## Results
The trained model successfully classifies Iris flower species with high accuracy.
